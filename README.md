# useITtoo
backend repository for school assignment creating a webshop and a control dashboard

controllers/ bevat de “actiebestanden” zoals AddToCartController.php, CartController.php, ProductController.php. Alle bestanden die formulier‑posts en routes afhandelen staan bij elkaar in controllers/.

classes/ of src/ bevat je domeinklassen (Product, OrderRow, ShoppingCart).

views/ bevat je HTML/PHP‑templates.
